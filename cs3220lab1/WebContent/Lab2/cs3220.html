Web Development with Eclipse and Tomcat

Tomcat

Download the latest Tomcat 8.5 binary release from the Apache Project. Note that there are several packages
available for download. You should download the ZIP file under the "Core" distribution. After downloading the
file, unzip it to a local directory, e.g. c:\apache-tomcat-8.5.x.

Eclipse

Download the latest Eclipse IDE for Java EE Developers ifrom Eclipse.org. Unzip it to a local directory, e.g.
c:\eclipse,then start up Eclipse by double-click on eclipse.exe under the Eclipse folder. If you are using Eclipse
for the first time, you will be asked to choose a folder to be Eclipse's workspace. Eclipse will store all your
projects in that folder.
